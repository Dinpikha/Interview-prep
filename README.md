# 🚀 Interview-prep

An AI-powered interview preparation platform designed to help candidates practice, learn, and improve through personalized guidance.

**Status:** 🟡 Active Development 

---

## 📖 Overview

Interview-prep is an ongoing project focused on making technical interview preparation more interactive and personalized.

Rather than simply providing questions, the platform combines AI assistance, mock interviews, resume analysis, and progress tracking into a single application.

The project is currently under active development and new features are continuously being added.

---

## ✨ Current Features

- 🤖 **AI Mentor** powered by a locally hosted LLM (LM Studio)
- 💬 Context-aware mentoring using custom system instructions

## 🧠 AI Mentor

The AI Mentor is designed to act as a personalized interview coach.

Instead of producing generic responses, it follows predefined instructions to guide users through concepts, encourage problem solving, provide hints, and explain mistakes in a conversational manner.

Current inference is powered locally using **LM Studio**, allowing experimentation with different open-source language models without relying on cloud APIs.
